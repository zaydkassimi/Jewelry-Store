# 💍 Jewelry Store

> A sleek online jewelry store built with HTML, CSS, and JavaScript — featuring product sorting, a dynamic shopping cart, and an admin dashboard for tracking buyer information and sales analytics.

---

## ✨ Features

- 🛍️ **Product Catalog** — Browse a curated collection of jewelry items
- 🔍 **Product Sorting** — Filter and sort items by category, price, or popularity
- 🛒 **Dynamic Shopping Cart** — Add, remove, and update items in real time
- 📊 **Admin Dashboard** — Track buyer information and view sales analytics
- 📱 **Responsive Design** — Optimized for all screen sizes

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- A local server (optional, e.g. VS Code Live Server)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/zaydkassimi/Jewelry-Store.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd Jewelry-Store
   ```

3. **Open in your browser**
   ```bash
   # Simply open index.html in your browser
   # Or use VS Code Live Server for best experience
   ```

## 📸 Pages

### 🏪 Storefront (`index.html`)
The main shopping page where customers can:
- Browse all jewelry products
- Sort and filter the catalog
- Add items to their cart
- Complete a purchase

### 🔧 Admin Panel (`admin.html`)
A secure dashboard where admins can:
- View buyer information
- Monitor sales data and analytics
- Manage product listings
